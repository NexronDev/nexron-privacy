Nexron Privacy Policy

This Privacy Policy explains how Nexron Discord Bot collects, uses, stores, and protects your information when you use our Discord bot.

Terms of Service Support Server Print Policy

Introduction

Nexron is committed to protecting your privacy. This Privacy Policy applies to all data collected by the Nexron Discord bot through Discord servers where it is present. By using Nexron, you consent to the data practices described in this policy.

Key Principles

  • We collect minimal data necessary for bot functionality
  • We never collect personal information beyond Discord IDs
  • All data is securely stored and protected
  • You have rights over your data
  • We are transparent about our practices

Our Privacy Commitment

Nexron's privacy-first approach

Minimal Data Storage

Only essential IDs stored: User IDs, Channel IDs, Guild IDs, and Category IDs.

Auto-Deletion

All logs content is automatically deleted after sending them.

No Personal Data

We never collect personal information. Your privacy is protected.

Data We Collect

Nexron collects only essential data required for proper functionality. We do NOT collect any personal information, messages, or sensitive data.

Data Type Purpose Storage Duration
Discord User IDs User identification for moderation actions, warnings, and bot functionality Until bot removal or data deletion request
Discord Server (Guild) IDs Server configuration storage and bot settings Until bot is removed from server
Discord Channel IDs Logging channels, and specific channel configurations Until configuration is changed or bot removed
Discord Role IDs Role assignment for moderation, verification, and permissions Until configuration is changed
Logs Content Temporary storage for moderation logs (AUTO-DELETED after sending) Deleted immediately after logging

What We NEVER Collect

  • ❌ Personal identification information (name, email, address, phone number)
  • ❌ Financial information (credit cards, bank details)
  • ❌ Private messages (except logs which are auto-deleted)
  • ❌ Voice chat recordings
  • ❌ IP addresses or location data

How We Use Your Data

All collected data is used exclusively for bot functionality and server management:

Primary Uses

  1. Moderation Features: User IDs for warnings, kicks, bans, and mutes
  2. Configuration Storage: Server IDs for saving bot settings and preferences
  3. Logging: Recording moderation actions and important events
  4. Verification: Role IDs for member verification systems
  5. Backup/Restore: Server structure data for raid protection

Data Sharing

We do NOT share, sell, or trade your data with third parties. The only exceptions are:

  • Discord API: Required for bot functionality (Discord's Privacy Policy applies)
  • Legal Requirements: If required by law or legal process
  • Security Protection: To prevent fraud, abuse, or security issues

Data Security

We implement industry-standard security measures to protect your data:

Security Measures

  • Encryption: All data in transit is encrypted using SSL/TLS
  • Access Control: Strict access limitations to stored data
  • Regular Audits: Security reviews of our systems
  • Secure Storage: Data stored on secure, reputable hosting services
  • Minimal Collection: Only essential data is collected and stored

Your Rights & Controls

You have control over your data. Here are your rights:

Data Access & Management

  • View Your Data: Request a copy of data we have about you
  • Data Deletion: Request deletion of your data
  • Opt-Out: Stop using the bot or remove it from your server
  • Correction: Request correction of inaccurate data

How to Exercise Your Rights

  1. Remove the Bot: Kick Nexron from your server to delete all server data
  2. Contact Us: Join our Support Server for data requests
  3. Server Owners: Can reset all bot data using the reset command

Data Retention

We retain data only as long as necessary:

Retention Periods

  • Active Server Data: Retained while the bot is in your server
  • Logs Content: Deleted immediately after sending is complete
  • Inactive Data: Purged after 6 months of inactivity

Children's Privacy

Nexron complies with Discord's age requirements. Discord requires users to be at least 13 years old. We do not knowingly collect data from children under 13.

Age Requirements

  • Discord's minimum age: 13 years old
  • We do not collect age information
  • Parents/guardians can request data deletion for underage users

  • Continued use after changes constitutes acceptance
